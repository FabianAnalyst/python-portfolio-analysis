# Python Portfolio Analysis

## Overview
This project performs a financial portfolio analysis using Python.
It compares Apple (AAPL) and the S&P 500 (^GSPC) from 2020 onwards.

The analysis includes:
- Daily returns
- Cumulative returns
- Volatility (risk)
- Risk vs Return comparison

## Skills Demonstrated
- Python (pandas, matplotlib, yfinance)
- Financial data analysis
- Risk and return modelling
- Data visualization
- Jupyter Notebook workflow

## Key Results
- Apple shows higher volatility than the S&P 500.
- Apple provides higher average return but with higher risk.
- The S&P 500 provides lower risk and lower return.

This project demonstrates practical financial data analysis and portfolio evaluation using Python.
